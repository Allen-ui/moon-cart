# 睡前逛逛 Moon Cart 小程序配置说明

这个目录是微信小程序原生 MVP 版本，不依赖 `web-view` 和业务域名。

## 需要你改的地方

1. 打开 `project.config.json`
   - 把 `appid` 改成你在微信公众平台拿到的小程序 AppID。

## 上传发布

1. 用微信开发者工具打开 `miniprogram-shell`。
2. 填入 AppID。
3. 本地预览确认首页、商品列表、详情、购物车、虚拟下单都能打开。
4. 点击右上角「上传」。
5. 到微信公众平台 -> 版本管理 -> 提交审核。
6. 审核通过后发布。

## 审核建议

H5 页面里要明确写：

「本产品为虚拟购物体验，不提供真实交易、不支付、不发货。」

避免被误判为真实电商交易小程序。
