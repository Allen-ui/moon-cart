# 就爱买小程序壳配置说明

这个目录是微信小程序 `web-view` 套壳版本，用来加载已经部署好的 H5 页面。

## 需要你改的地方

1. 打开 `project.config.json`
   - 把 `appid` 改成你在微信公众平台拿到的小程序 AppID。

2. 打开 `pages/index/index.js`
   - 把 `h5Url` 改成你的线上 H5 地址，例如：
     `https://justbuy.example.com`

## 微信后台需要配置

在微信公众平台进入：

设置 -> 开发设置 -> 业务域名

把你的 H5 域名添加进去。

注意：

- 必须是 HTTPS。
- 通常需要已备案域名。
- 需要按微信后台提示下载校验文件，并放到域名根目录。
- 个人主体小程序可能无法使用 `web-view`，以后台实际提示为准。

## 上传发布

1. 用微信开发者工具打开 `miniprogram-shell`。
2. 填入 AppID。
3. 本地预览确认能打开 H5。
4. 点击右上角「上传」。
5. 到微信公众平台 -> 版本管理 -> 提交审核。
6. 审核通过后发布。

## 审核建议

H5 页面里要明确写：

「本产品为虚拟购物体验，不提供真实交易、不支付、不发货。」

避免被误判为真实电商交易小程序。
